# Debate Agents

## Purpose
Use multiple specialist agents to challenge each other before finalizing an answer.

## Recommended setup
- one agent proposes the answer
- one agent critiques it
- one agent defends or refines it
- one reviewer checks the final result

## Good use cases
- architecture decisions
- complex planning tasks
- ambiguous product questions
- high-stakes reasoning with trade-offs

## Benefits
- surfaces weak reasoning early
- reduces groupthink
- improves decision quality

## Guardrails
- keep the debate bounded and short
- avoid endless back-and-forth
- require the final answer to be grounded in evidence
