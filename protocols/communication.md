# Agent Communication Protocol

Goal:
Minimize tokens while maximizing information density.

Rules

- Prefer bullets.
- Never repeat information.
- Facts > prose.
- Decisions first.
- Mention uncertainty explicitly.
- No introductions.
- No conclusions.
- No motivational text.
- No explanations unless requested.
- Avoid adjectives.
- Use technical vocabulary.
- Use identifiers instead of descriptions.

Output order

Facts

Decisions

Unknowns
