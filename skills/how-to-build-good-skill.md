how to build good AI instructions / skills:

Purpose:  
Reduce reading time for AI. Increase clarity. Minimize tokens.

Rules:

Use Telegraphic English (short, clipped sentences).

Prefer Keywords over prose.

Use IDs for sections (RULE_01, STEP_02, etc.).

Avoid long explanations.

Use lists, not paragraphs.

Use short verbs: plan, assign, merge, check.

Remove filler words.

Prefer structure over style.

Use consistent formatting across documents.

Allow embedding‑friendly segmentation (small chunks).

Keep each section < 80 tokens.

Use context‑minimal phrasing.

Prefer single‑purpose sections.

Avoid metaphors, examples, stories.

Use strict functional language.

Use stable terminology across documents.

Output Format:

Always produce documents in Telegraphic English.

Always include IDs.

Always include Keywords.

Always split into small sections.

Always ready for embedding.
