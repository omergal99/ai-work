# Session History

## Purpose
This folder preserves a durable record of important AI-session decisions and progress.

## Required Process
- After each meaningful session, create a dated folder under .ai/ops/history/.
- Inside it, create README.md with:
  - summary
  - decisions
  - open questions
  - follow-up actions
- Link to the relevant main documents when the change affects them.

## Naming Convention
Use the format: YYYY-MM-DD or YYYY-MM-DD-shortname
